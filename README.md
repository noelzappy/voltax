# Voltax Monorepo

**Voltax** is a polyglot open-source payment SDK designed to unify African payment infrastructure.

## Structure

- **`packages/node`**: The core TypeScript/Node.js SDK.

## Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Build Packages**:
   ```bash
   npm run build --workspaces
   ```

## Contributing

We welcome contributions! Please check the `packages/` directory for the specific language implementation you want to contribute to.
